<h1 align="center">Arrays</h1>

<h3>Vetores</h3>

```
tipo nome[tamanho];
nome[índice] = valor;
```
<h3>Matrizes</h3>

```
tipo nome[linhas][colunas];
nome[indiceLinha][indiceColuna] = valor;
```
<h3>Alocação Dinâmica</h3>

```
vetor = malloc(sizeof(tipo) * tamanho);
matriz = malloc(sizeof(tipo) * linhas * colunas);
```
